## 1. Clone the Project
Bash

### Clone the repository
git clone https://github.com/skImranSabuj/sport-app.git

Navigate into the project directory
cd sport-app

## 2. Install Dependencies
Using your preferred package manager:

npm install
or
yarn install

##3. Environment Configuration
Create a .env file in the root directory and add your API base URL:

EXPO_PUBLIC_API_BASE_URL=https://smartb.com.au/soc-api

## 4. Running the App
Start the Expo development server:

```npx expo start```
Once the server is running:

Android: Scan the QR code using the Expo Go app.

iOS: Scan the QR code with your Camera app.

Emulators: Press a for Android or i for iOS to open the app in a simulator.
